# 🧭 Frontend Roadmap Dashboard (на русском)

Интерактивный дашборд для изучения и отслеживания прогресса по фронтенд-технологиям (React + TypeScript).

## 🚀 Возможности
- Отмечай завершённые темы;
- Добавляй свои темы и заметки;
- Фильтруй по уровню (Junior, Middle, Senior);
- Экспортируй/Импортируй прогресс в JSON;
- Ссылки на лучшие учебные ресурсы;
- Локальное сохранение (localStorage).

## ⚙️ Установка и запуск

```bash
npm install
npm run dev
```
Открой [http://localhost:5173](http://localhost:5173)

## 🧩 Технологии
- React + Vite
- Tailwind CSS
- Framer Motion
